## О репозитории

**Краткое описание:**
Данный репозиторий создан для выполнения лабораторных/практических работ по дисциплине «[Информатика]». Здесь будут размещаться отчеты, исходный код программ, скриншоты и сопутствующие материалы, необходимые для успешной сдачи заданий. Основная цель — систематизация учебных проектов и демонстрация progress'a в ходе семестра.

## Планируемые материалы

В рамках курса планируется разместить следующие файлы и папки:

*   **/lab_01** - Лабораторная работа №1: "Основы работы со средой разработки".
    *   `report.docx` / `report.pdf` - отчет.
    *   `src/` - исходный код первой программы.
*   **/lab_02** - Лабораторная работа №2: "Работа с функциями и массивами".
    *   `algorithm_description.md` - описание алгоритма.
    *   `code_example.py` (или `.cpp` / `.java`) - пример кода.
*   **/lab_03** - Лабораторная работа №3: "Объектно-ориентированное программирование".
    *   `diagrams/` - папка с UML-диаграммами.
    *   `classes_implementation/` - реализация классов.
*   **/project** - Курсовой проект (или итоговая работа).
    *   `docs/` - пояснительная записка.
    *   `presentation/` - презентация.
    *   `demo/` - скриншоты или видео работы приложения.
*   **README.md** - данный файл с описанием репозитория.
*   **.gitignore** - файл для исключения временных файлов и сборок из репозитория.
