## 1. Тема работы
Знакомство с системой контроля версий Git и платформой GitHub. Создание и оформление репозитория для учебных проектов.

## 2. Цель работы
Освоить базовые навыки работы с Git (инициализация репозитория, создание коммитов, добавление файлов) и платформой GitHub (создание удаленного репозитория, связывание с локальным, редактирование файлов через веб-интерфейс). Научиться оформлять файлы в формате Markdown (.md) для структурированного описания проектов и отчетов.
